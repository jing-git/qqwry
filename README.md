* HELP

```
usage: qqwry [-h] [-q QUERY [QUERY ...] | --query-file QUERY_FILE
             [QUERY_FILE ...] | --dump] [--update-db] [--quiet] [-f DBPATH]
             [-o OUTPUT]

qqwry is an IP address lookup process, for QQWry(cz88.net).

optional arguments:
  -h, --help            show this help message and exit
  -q QUERY [QUERY ...]  lookup ip.
  --query-file QUERY_FILE [QUERY_FILE ...]
                        lookup ip from files
  --dump                dump all ip information to a file.
  --update-db           update qqwry.dat
  --quiet               quiet mode.
  -f DBPATH, --dbpath DBPATH
                        the path of qqwry.dat
  -o OUTPUT, --output OUTPUT
                        save output of ip info to a file
```
